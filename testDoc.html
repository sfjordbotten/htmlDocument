Basic HTML

This section shows basic html functionality.

Paragraph Text

Text can be added using the addText method, which will be wrapped in a paragraph (<p>) tag. You can also include html tags in the text, such as <b>bold</b> or <i>italics</i>.

By default, special characters in the text will be escaped, except for html tags allowed inside a paragraph. If you want to limit allowed html tags, you can use the "keepTags" parameter to specify which tags to allow. If you want to prevent this "smart" behaviour, you can use the allAsText or allAsIs paramters. The allAsText method will escape all special characters (including html tags), while the allAsIs method will not escape any special characters.

Lists

You can add ordered and unordered lists using the addOrderedList and addUnorderedList methods. These methods take a list of items to include in the list. Each item can be a string or a block of html code. You can also include nested lists by including a list as an item in the main list.

The following is an example of an ordered list:

  1. First item
  2. Second item
    1. 2a
    2. 2b
  3. Third item

The following is an example of an unordered list:

  • First item
  • Second item
  • Third item
    • Subitem 1
    • Subitem 2

The following is an example of an ordered list with a nested unordered list. Note that the unordered list is included as part of the second item in the ordered list (rather than as its own item):

  1. First item
  2. Second item
    • Subitem, unorderd
  3. Third item
